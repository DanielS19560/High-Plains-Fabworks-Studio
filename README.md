# High Plains Fabworks Studio

This repository is being converted to a private canonical brand-assets home.

The initial public asset upload has been withdrawn from `main` while repository visibility is updated.

## Recommended Setup

- Keep the full source-of-truth brand pack in this repo as `private`
- Publish only approved Canva/web export assets from a separate public location

## Why

Canva can import from direct public HTTPS URLs, but it cannot fetch from a private GitHub repository unless files are manually uploaded.
